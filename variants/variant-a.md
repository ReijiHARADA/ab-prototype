# Variant A

## 役割 / Intent

ベースラインまたはコントロール条件。  
기준선 또는 통제 조건.

## 内容 / Content

不特定多数の選択を示す一般的なソーシャルプルーフを表示する。  
불특정 다수의 선택을 보여주는 일반적인 소셜 프루프를 표시한다.

文言例:

- 今人気  
- 多くの人が見ています  
- 売れ筋商品

## UI 差分 / UI differences

- CTA 近くに短いラベルまたは補助テキストを表示する  
  CTA 근처에 짧은 라벨 또는 보조 텍스트를 표시한다
- レイアウトは Variant B とそろえ、文言の種類だけを変える  
  레이아웃은 Variant B와 맞추고 문구 종류만 바꾼다

## 期待するユーザー行動 / Expected user behavior

一般的な安心感や人気認知は得られるが、個人との関連性は限定的であると想定する。  
일반적인 안심감과 인기 인지는 얻을 수 있지만, 개인과의 관련성은 제한적일 것으로 본다.
