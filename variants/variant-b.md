# Variant B

## 役割 / Intent

処置条件。  
처치 조건.

## 内容 / Content

ユーザーに近い集団を示すソーシャルプルーフを表示する。  
사용자와 유사한 집단을 보여주는 소셜 프루프를 표시한다.

文言例:

- 20代女性に人気  
- この地域で人気  
- 同じ目的の人に人気

## UI 差分 / UI differences

- CTA 近くに短いラベルまたは補助テキストを表示する  
  CTA 근처에 짧은 라벨 또는 보조 텍스트를 표시한다
- 配置と視認性は Variant A とそろえ、集団の近さを示す内容だけを変える  
  배치와 가시성은 Variant A와 맞추고, 집단의 유사성을 보여주는 내용만 바꾼다

## 期待するユーザー行動 / Expected user behavior

自分に関係のある情報だと受け取られやすくなり、信頼感、説得力、クリックやカート追加の意図が高まることを想定する。  
나와 관련 있는 정보로 받아들여질 가능성이 높아져 신뢰감, 설득력, 클릭 및 장바구니 추가 의도가 높아질 것으로 본다.
